⚠ 가격 해석 시 유의사항: H100 단가 행은 각사의 대표 인스턴스 온디맨드 리스트 가격이며, 제공 방식(VM vs Baremetal)이 다르므로 단순 비교 불가. Oracle $10/GPU/hr은 Baremetal(오버헤드 0%) + NVMe 61.4TB + Egress 10TB 무료가 포함된 가격이고, AWS·GCP $3~4는 VM 기준(오버헤드 5~10%). 동일 스펙(Baremetal + InfiniBand) 비교 시 Azure ND96isr $12.29 vs Oracle $10.00으로 Oracle이 약 19% 저렴.

TCO(총소유비용) 기준: 대규모 AI 학습 워크로드(8×GPU + 스토리지 + Egress 포함) 벤치마크에서 Oracle이 AWS 대비 38%, Azure 대비 32% 저렴. Oracle의 Egress 비용은 하이퍼스케일러의 약 1/10 수준(10TB 무료 후 $0.0085/GB vs AWS $0.09/GB)이며, 블록 스토리지도 AWS 대비 68%, Azure 대비 79% 저렴.

ClusterMAX™: SemiAnalysis가 Security, Networking, Storage, Orchestration, Reliability 등 10개 항목을 종합 평가한 GPU 클라우드 등급(5단계). v2.0(2025.11) 기준 37개 클라우드 평가. Platinum은 CoreWeave가 유일.
